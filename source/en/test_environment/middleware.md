# Related Hardware
* SF32LB52-DevKit-Core-3p3 (on-board chip models SF32LB52BU56 / SF32LB52JUD6). For usage, refer to the [SF32LB52-DevKit-Core-3p3 Development Board User Guide](http://10.21.10.172:6600/devkits/sf32lb52x/SF32LB52-DevKit-Core-3p3.html).
* Agilent 66319D DC power supply.
* Keysight 34465A digital ammeter.
* Serial (UART) board.
* Android phone, install the LightBlue App (com.punchthrough.lightblueexplorer_LightBlue_1.9.3.apk, available from the Google Play Store) or install a BLE Debug Assistant (available from the built-in Android app store).
